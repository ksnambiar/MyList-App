# Simple List App

## How to install ?

1. git clone the repository
```bash
git clone https://github.com/ksnambiar/MyList-App.git
```
2. cd into the MyList-App folder
```bash
cd MyList-App
```
3. Install all the required npm packages
```bash
npm install
```

## How to run

run the application using the following command
```bash
npm start
```
